# 🏠 Smart Home Automation System

A comprehensive SQL Server database system for managing IoT devices, automation rules, energy monitoring, and real-time alerts.

## 📁 Project Files

| File | Description |
|------|-------------|
| `Smart_Home_Automation_Database.sql` | Complete database script (tables, indexes, stored procedures, views, sample data) |
| `Smart_Home_Documentation.md` | Technical documentation with ER diagram and backup strategy |
| `Demo_Script.sql` | Step-by-step live demo script for presentation |
| `presentation.html` | Interactive presentation slides (open in browser) |
| `ER_Diagram.png` | Entity-Relationship diagram |

## 🏗️ Database Features

- **5 Tables**: Users, Devices, AutomationRules, EnergyUsage, Alerts
- **6 Indexes**: Optimized for real-time IoT query patterns
- **5 Stored Procedures**: Secure CRUD operations with error handling
- **4 Views**: Real-time reporting dashboards
- **Backup & Recovery Plan**: Full Recovery Model with point-in-time restore

## 🚀 How to Run

1. Open **SQL Server Management Studio (SSMS)**
2. Execute `Smart_Home_Automation_Database.sql` to create the database
3. Use `Demo_Script.sql` to test all features

## 📊 Presentation

Open `presentation.html` in any web browser. Use arrow keys to navigate slides. Press `F` for fullscreen.
